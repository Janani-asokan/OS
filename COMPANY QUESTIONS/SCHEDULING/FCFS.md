🎯 How To Explain This Code in Interview

You should say:

FCFS is a non-preemptive scheduling algorithm. Processes are executed in order of arrival. Waiting time is calculated as the difference between current CPU time and arrival time. Turnaround time is the sum of waiting time and burst time.

Then explain:

Time Complexity → O(n log n) (because of sorting)

No starvation

Suffers from convoy effect

🔥 Important Concepts You Should Mention

If interviewer pushes deeper, say:

It is non-preemptive

Average waiting time can be high

Convoy effect happens when a long process blocks short ones

Suitable for batch systems, not interactive systems
